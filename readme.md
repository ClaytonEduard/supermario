## 🚀 Instalação

1. Clone este repositório:

   ```bash
   git clone https://github.com/ClaytonEduard/supermario.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd supermario
   ```
3. Abra o arquivo index.html no seu navegador para visualizar o projeto.
